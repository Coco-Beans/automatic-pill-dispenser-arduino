# Automatic Pill Dispenser using Arduino

## Overview
This project implements an automatic pill dispenser system using Arduino. The system reminds users to take medicine and dispenses pills automatically using a servo motor.

The LCD screen displays reminders, while a buzzer alerts the user when it is time to take medication.

## Features
• Automatic pill dispensing  
• LCD display for reminders  
• Audible alert using buzzer  
• Servo motor controlled pill release  
• Simple user interaction using push button  

## Components Used
- Arduino Uno
- Servo Motor
- 16x2 LCD Display
- Buzzer
- Push Button
- Potentiometer
- Breadboard and jumper wires

## Working Principle
1. The Arduino controls the entire system.
2. The LCD displays the current status or reminder message.
3. When the scheduled time arrives, the buzzer alerts the user.
4. The servo motor rotates to release a pill.
5. The system waits for the next cycle.
